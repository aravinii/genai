# Course materials

This folder contains the results and materials from **two courses** used to build the final project: a **multi-agent real estate assistant** (corretor imobiliário) flow.

For more details on each course and its contents, see the README and notebooks inside each subfolder:

- **`generative_ai/`** — Generative AI (prompting, RAG, function calling, agents).
- **`agentic_ai/`** — Agentic AI (multi-step agents, assignments).
